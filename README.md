Railway Chatbot Website
Description

This project is a simple railway information website integrated with an AI chatbot built using IBM Watsonx Assistant.
The website allows users to view train schedules, access a ticket booking interface, and interact with a chatbot for basic railway-related assistance.

Technologies Used

HTML

CSS

IBM Watsonx Assistant

Files

index.html – Main webpage

style.css – Website styling

How to Run

Open index.html in any web browser to view the website.
